# User Manual — SST

## Purpose

End-user instructions for Sales, TA, HR, and Leadership.

## Audience

Business operators.

## Scope

MVP workflows replacing Excel.

## Definitions

See product glossary in docs README.

---

## Getting started

1. Open SST URL  
2. Sign in with your email  
3. Use left navigation  

## Sales — create a requirement

1. Go to **Requirements** → **New**  
2. Enter client, role/skill, positions, priority, sales owner  
3. Save  
4. Assign TA owner and TA handoff date when ready  
5. Watch SLA badge (Green/Amber/Red)  

## TA — manage candidates

1. Open requirement or **Candidates**  
2. **Add candidate** with mobile/email  
3. Watch duplicate warnings  
4. Update stage/feedback  
5. Toggle **Selected** when client accepts  

## HR — offers & onboarding

1. Open selected candidate → **Create offer**  
2. Enter CTC/rate, expected DOJ, release offer  
3. On **Accepted** → **Start onboarding**  
4. Track docs/BGV  
5. Set actual DOJ and mark **Joined**  

## Leadership — dashboard

1. Open **Dashboard**  
2. Filter by owner/client/priority/dates  
3. Review KPIs, stage mix, RAG, escalations  

## Tips from Excel

- Blue fields were editable inputs → form fields  
- Gray formula fields → read-only badges  
- Filters mirror Excel dashboard filter bar
